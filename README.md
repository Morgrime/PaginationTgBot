# Бот читалка для тг
Проект бота читалки для телеграмма

# Функционал
- Пока что только поддерживает 1 книгу (/book/book.txt)
- Поддержка закладок (в том числе и переход к ним)
- Есть менюшка для удобной работы
- Переход по страницам
